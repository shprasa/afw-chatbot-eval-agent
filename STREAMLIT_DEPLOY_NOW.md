# Deploy the AFW platform on Streamlit Cloud

1. Go to https://share.streamlit.io (GitHub login as `shprasa`)
2. **Create app** → `afw-chatbot-eval-agent` / `main` / `streamlit_app.py`
3. Add secrets: `GITHUB_TOKEN`, `GITHUB_OWNER`, `GITHUB_REPO`, `GITHUB_BRANCH`
4. Deploy

You get **Dashboard** + **Eval Agent** in one URL. No local install needed.

After eval: **Eval Agent** → **Save to GitHub repo** → **Dashboard** refreshes.
